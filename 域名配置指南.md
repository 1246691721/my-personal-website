# 域名配置指南

## 购买域名

### 推荐域名注册商（国内）
1. **阿里云（万网）**：https://wanwang.aliyun.com
   - 价格：约 50-100 元/年（.com 域名）
   - 支持备案，国内访问快
   
2. **腾讯云**：https://dnspod.cn
   - 价格：约 50-100 元/年
   - 与腾讯服务集成好

3. **GoDaddy**（国外）：https://www.godaddy.com
   - 价格：约 $10-15/年
   - 无需备案，但国内访问可能较慢

### 购买步骤
1. 注册账号并登录
2. 搜索想要的域名（如：`lizhuoran.com` 或 `lizhuoran.cn`）
3. 选择域名并加入购物车
4. 完成支付

## 域名配置（以 GitHub Pages 为例）

### 方法一：GitHub Pages + 自定义域名

1. **在 GitHub 仓库中设置**：
   - 进入你的仓库
   - Settings > Pages
   - 在 "Custom domain" 输入框输入你的域名（如：`www.lizhuoran.com`）
   - 勾选 "Enforce HTTPS"

2. **在域名注册商配置 DNS**：
   
   **如果使用根域名（如 lizhuoran.com）：**
   ```
   类型：A
   主机记录：@
   记录值：185.199.108.153
   记录值：185.199.109.153
   记录值：185.199.110.153
   记录值：185.199.111.153
   ```
   
   **如果使用 www 子域名（如 www.lizhuoran.com）：**
   ```
   类型：CNAME
   主机记录：www
   记录值：YOUR_USERNAME.github.io
   ```
   
   **同时配置根域名（推荐）：**
   ```
   类型：A
   主机记录：@
   记录值：185.199.108.153
   记录值：185.199.109.153
   记录值：185.199.110.153
   记录值：185.199.111.153
   ```

3. **在仓库根目录创建 CNAME 文件**：
   - 文件名：`CNAME`（无扩展名）
   - 内容：你的域名（如：`www.lizhuoran.com` 或 `lizhuoran.com`）
   - 提交并推送到 GitHub

### 方法二：Vercel + 自定义域名（推荐，更简单）

1. **部署到 Vercel**：
   - 访问 https://vercel.com
   - 用 GitHub 账号登录
   - 点击 "Add New Project"
   - 导入你的 GitHub 仓库
   - 自动部署完成

2. **添加自定义域名**：
   - 在 Vercel 项目页面，进入 Settings > Domains
   - 输入你的域名（如：`lizhuoran.com`）
   - 按照提示配置 DNS 记录

3. **配置 DNS**（在域名注册商）：
   ```
   类型：CNAME
   主机记录：@（或 www）
   记录值：cname.vercel-dns.com
   ```
   
   或者使用 A 记录：
   ```
   类型：A
   主机记录：@
   记录值：76.76.21.21
   ```

## DNS 配置示例（阿里云）

1. 登录阿里云控制台
2. 进入 **域名** > **解析**
3. 找到你的域名，点击 **解析设置**
4. 添加记录：
   - **记录类型**：A 或 CNAME
   - **主机记录**：@ 或 www
   - **记录值**：根据上面的说明填写
   - **TTL**：600（默认）

## 验证配置

DNS 配置后，需要等待 **10 分钟到 48 小时** 生效（通常 1-2 小时）。

验证方法：
```bash
# 在终端执行（把域名替换成你的）
ping www.lizhuoran.com
# 或
nslookup www.lizhuoran.com
```

## 备案（仅限国内域名 + 国内服务器）

⚠️ **重要**：
- 如果使用 **GitHub Pages** 或 **Vercel**（国外服务器），**不需要备案**
- 如果使用国内服务器（如阿里云、腾讯云），需要备案
- `.com`、`.cn` 等域名都可以不备案使用国外服务器

## 常见问题

### DNS 不生效？
- 等待更长时间（最长 48 小时）
- 检查 DNS 记录是否正确
- 清除浏览器缓存
- 使用不同网络环境测试

### HTTPS 证书？
- GitHub Pages 和 Vercel 都自动提供免费 SSL 证书
- 配置域名后会自动启用 HTTPS

### 域名解析到多个服务？
- 可以配置多个子域名：
  - `www.lizhuoran.com` → 主网站
  - `blog.lizhuoran.com` → 博客（如果需要）

## 推荐方案

**最简单方案**：
1. 购买域名（阿里云或腾讯云）
2. 使用 Vercel 部署（自动 HTTPS）
3. 在 Vercel 添加自定义域名
4. 按照 Vercel 提示配置 DNS

这样就不需要手动创建 CNAME 文件，Vercel 会自动处理！
