# 📝 如何添加新的博客文章

> 这是一份超级简单的教程，即使是编程小白也能轻松上手！

---

## 方法一：使用模板文件（推荐！最简单）

### 步骤 1️⃣ 复制模板
1. 找到 `blog-template.html` 文件
2. 右键 → 复制
3. 粘贴到同一文件夹
4. 重命名为你的文章名，例如：`my-first-blog.html`

### 步骤 2️⃣ 修改内容
用任意文本编辑器打开文件（推荐使用 VS Code 或 Cursor），找到以下位置修改：

| 要修改的内容 | 在文件中搜索 | 修改方法 |
|-------------|-------------|---------|
| 文章标题 | `文章标题` | 替换成你的标题 |
| 日期 | `2024年12月4日` | 改成今天日期 |
| 分类 | `💭 思考` | 改成适合的分类 |
| 正文内容 | `这里是文章的开头段落` | 写你的内容 |
| 标签 | `#ENFP` | 改成你的标签 |

### 步骤 3️⃣ 添加到博客列表
打开 `blog.html`，找到 `<div class="blog-list">`，复制一个 `<article class="blog-item">` 块，修改成你的新文章信息。

## 方法二：直接在 blog.html 中添加文章卡片

### 复制这段代码到 blog.html 的 `<div class="blog-list">` 里面：

```html
<article class="blog-item reveal">
  <div class="blog-item-image" style="background: linear-gradient(135deg, #FF6B6B, #FFE66D);">
    💭
  </div>
  <div class="blog-item-content">
    <div class="blog-item-meta">
      <span class="blog-item-category">思考</span>
      <span class="blog-item-date">2024年12月4日</span>
    </div>
    <h3 class="blog-item-title">你的文章标题</h3>
    <p class="blog-item-excerpt">
      这里写文章的简短摘要，大约2-3句话，让读者知道这篇文章讲什么...
    </p>
    <div class="blog-item-tags">
      <span class="blog-tag">#标签1</span>
      <span class="blog-tag">#标签2</span>
    </div>
  </div>
</article>
```

### 修改说明：
- **背景颜色**：改 `#FF6B6B, #FFE66D` 为你喜欢的颜色
- **图标**：改 `💭` 为适合的 emoji
- **分类**：改 `思考` 为你的分类
- **日期**：改成今天日期
- **标题**：改成你的文章标题
- **摘要**：写文章简介

## 📝 写文章的小技巧

### 常用 HTML 标签

```html
<!-- 段落 -->
<p>这是一个段落的内容...</p>

<!-- 小标题 -->
<h2>🌟 这是小标题</h2>

<!-- 引用 -->
<blockquote>"这是引用的话"</blockquote>

<!-- 无序列表 -->
<ul>
  <li>第一点</li>
  <li>第二点</li>
</ul>

<!-- 有序列表 -->
<ol>
  <li>第一步</li>
  <li>第二步</li>
</ol>

<!-- 插入图片 -->
<img src="images/your-image.jpg" alt="图片描述">

<!-- 链接 -->
<a href="https://example.com">点击这里</a>

<!-- 加粗 -->
<strong>加粗的文字</strong>

<!-- 斜体 -->
<em>斜体的文字</em>
```

## ❓ 常见问题

### Q: 我修改后网页没变化？
A: 按 `Ctrl+Shift+R`（Mac 用 `Cmd+Shift+R`）强制刷新页面

### Q: 怎么添加图片？
A: 
1. 把图片放到 `images` 文件夹
2. 在 HTML 中用 `<img src="images/图片名.jpg">` 插入

### Q: 格式乱了怎么办？
A: 检查是否漏了结束标签（如 `</p>`、`</div>`）
