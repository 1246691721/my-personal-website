# 李卓然的个人网站

一个展示个人经历、博客和作品集的个人网站。

## 功能特点

- 🏠 **主页**：个人介绍和基本信息
- 📝 **博客**：分享思考和感悟
- 💼 **作品集**：展示项目和研究
- 👥 **回忆录**：记录与朋友们的珍贵回忆
- 💌 **留言墙**：朋友们可以留下祝福和留言
- 📧 **联系方式**：便捷的联系表单

## 技术栈

- HTML5 / CSS3 / JavaScript
- 响应式设计
- Supabase（留言墙数据存储）

## 部署

网站已部署到 GitHub Pages / Vercel，可通过域名访问。

## 本地运行

直接在浏览器中打开 `index.html` 即可查看。
