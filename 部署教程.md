## 方法一：Vercel 部署（最简单，5分钟）

### 第 1 步：注册 Vercel 账号

1. 打开浏览器，访问 **https://vercel.com**
2. 点击右上角 **Sign Up**（注册）
3. 选择 **Continue with Email**（邮箱注册）或 **Continue with GitHub**
4. 填写邮箱、设置密码，完成注册
5. 去邮箱点击验证链接

### 第 2 步：上传网站

1. 登录后，点击 **Add New...** → **Project**
2. 在页面下方找到 **Import Third-Party Git Repository** 
3. 或者更简单：**直接把 `自制网页` 文件夹拖拽到网页上！**

```
📁 自制网页  ──拖拽到──▶  🌐 Vercel 网页
```

4. 等待 30 秒左右，自动部署完成

### 第 3 步：获取网址

部署成功后，你会得到一个网址，类似：
```
https://zi-zhi-wang-ye.vercel.app
```

✅ **完成！** 把这个网址分享给任何人，他们都能看到你的网站了！

### 💡 自定义域名（可选）

如果你想要自己的域名（如 lizhuo-ran.com）：
1. 在域名注册商购买域名（阿里云、腾讯云等）
2. 在 Vercel 项目设置中添加域名
3. 按提示配置 DNS

## 方法二：GitHub Pages 部署（推荐长期使用）

### 准备工作

1. 下载 **GitHub Desktop**：https://desktop.github.com
2. 注册 **GitHub 账号**：https://github.com

### 第 1 步：注册 GitHub

1. 打开 **https://github.com**
2. 点击 **Sign up**
3. 输入邮箱、创建密码、选择用户名
4. 完成验证，去邮箱确认

### 第 2 步：安装 GitHub Desktop

1. 打开 **https://desktop.github.com**
2. 下载 Mac 版本
3. 安装并打开
4. 点击 **Sign in to GitHub.com**
5. 登录你的 GitHub 账号

### 第 3 步：创建仓库

**在 GitHub Desktop 中：**

1. 点击 **File** → **New Repository**
2. 填写信息：
   - **Name**（仓库名）：`my-website` 或 `lizhuo-ran.github.io`（推荐后者）
   - **Local Path**：选择你的 `自制网页` 文件夹的**上一级目录**
   - ✅ 勾选 **Initialize this repository with a README**
3. 点击 **Create Repository**

### 第 4 步：添加网站文件

1. 打开 Finder，进入 `自制网页` 文件夹
2. 全选所有文件（Cmd + A）
3. 复制（Cmd + C）
4. 进入 GitHub Desktop 创建的仓库文件夹
5. 粘贴（Cmd + V）

### 第 5 步：提交并推送

**在 GitHub Desktop 中：**

1. 左侧会显示所有新增的文件
2. 在左下角 **Summary** 框中输入：`首次上传网站`
3. 点击 **Commit to main**（提交）
4. 点击右上角 **Push origin**（推送到 GitHub）

### 第 6 步：开启 GitHub Pages

1. 打开浏览器，登录 **https://github.com**
2. 进入你刚创建的仓库
3. 点击 **Settings**（设置）
4. 左侧菜单找到 **Pages**
5. 在 **Source** 下拉框选择 **main** 分支
6. 点击 **Save**
7. 等待 1-2 分钟

### 第 7 步：访问网站

你的网站地址是：
```
https://你的用户名.github.io/仓库名/
```

例如：
```
https://lizhuo-ran.github.io/my-website/
```

如果仓库名是 `你的用户名.github.io`，则网址是：
```
https://你的用户名.github.io/
```

✅ **完成！**

## 如何更新网站

### Vercel 更新方法

1. 在本地电脑修改文件
2. 重新把整个文件夹拖拽到 Vercel
3. 自动覆盖更新

### GitHub Pages 更新方法

1. 在本地电脑修改文件
2. 打开 **GitHub Desktop**
3. 左侧会显示修改的文件
4. 在 **Summary** 输入修改说明，如：`添加新博客`
5. 点击 **Commit to main**
6. 点击 **Push origin**
7. 等待 1-2 分钟，网站自动更新

```
修改文件 → 打开 GitHub Desktop → Commit → Push → 完成！
```

## 常见问题

### Q1: 网站打开是空白的？
**A:** 检查 `index.html` 是否在根目录，不能在子文件夹里。

### Q2: 图片不显示？
**A:** 
- 检查图片文件名是否正确（区分大小写）
- 确保图片已经上传

### Q3: 样式丢失？
**A:** 检查 `css/style.css` 路径是否正确。

### Q4: GitHub Pages 显示 404？
**A:** 
- 等待几分钟再试
- 检查 Settings → Pages 是否正确配置
- 确保有 `index.html` 文件

### Q5: 如何绑定自己的域名？
**A:** 
1. 购买域名（阿里云/腾讯云）
2. GitHub：在仓库根目录创建 `CNAME` 文件，内容为你的域名
3. 在域名管理处添加 CNAME 记录指向 `你的用户名.github.io`
